Repositorio do projeto de chat ia, que integra varias IAs em um site, tirando a nessecidade de ter que abrir seus respequetivos sites, atualmente só suporta o gemini.<br>
Guia dos projetos <br><br>
api_gemini<br>
 Primeira versão do progeto chat, primeira tentativa de rodar A API do gemini.<br><br>
api_gemini_docker<br>
 Segunda versão do projeto chat, está funcionando com o minimo para funcionar usa flask para o backend e vita para o frontend. Atualmente está quebrada.<br><br>
PDF_translator<br>
 projeto auxiliado com ia, para traduzir pdfs, progeto pessoal. ainda não esta desenvolvida.<br><br>
api_ia_web<br>
Projeto ia, feito com javascript, node, docker, react, vite.
